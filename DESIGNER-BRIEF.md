## 🖼️ Требования к изображениям

### Hero секции
- **Размер:** 1920×1080px минимум
- **Формат:** WebP (с JPEG fallback)
- **Стиль:** Тёмные, драматичный свет, сцена/театр
- **Обработка:** Затемнение 40-60% для читаемости текста

### Портреты
- **Kateryna:** Профессиональные, театральные, уверенные
- **Клиенты:** Бизнес casual, разнообразие, в действии

### Карточки/thumbnails
- **Размер:** 400×300px (4:3)
- **Формат:** WebP
- **Весe:** < 50KB каждая

### Иконки
- **Стиль:** Линейные, 24×24px base
- **Stroke:** 1.5-2px
- **Источник:** Lucide Icons или аналог

## 🔧 Как редактировать

### Цвета
Файл: `css/style.css` → секция `:root`
```css
:root {
  --color-primary: #3C5072;    /* <-- меняй здесь */
  --color-accent: #e6b264;
  /* ... */
}
```

### Шрифты
Файл: `index.html` → `<head>` → Google Fonts link
```html
<link href="https://fonts.googleapis.com/css2?family=...&display=swap">
```

### Контент страниц
- `index.html` — главная
- `about.html` — о Катерине
- `masterclass.html` — мастер-классы
- `corporate.html` — B2B
- `coaching.html` — 1-на-1
- `studio.html` — Studio Labs
- `courses.html` — онлайн курсы
- `contact.html` — контакт
- `payment.html` — оплата

### Добавление изображений
1. Сохрани в `images/` в формате WebP
2. Оптимизируй: < 100KB для hero, < 50KB для карточек
3. Добавь в HTML:
```html
<img src="images/photo.webp" alt="Описание" loading="lazy" width="400" height="300">
```

## ⚠️ На что обращать внимание

### ✅ DO (делать)
- Сохранять тёмную тему как основу
- Использовать accent gold для CTA
- Поддерживать читаемость (контраст 4.5:1+)
- Добавлять `loading="lazy"` к изображениям
- Тестировать на мобильных (50%+ трафика)
- Использовать CSS переменные для цветов

### ❌ DON'T (не делать)
- Яркие неоновые цвета
- Белый/светлый фон
- Мелкий текст (< 14px на mobile)
- Тяжёлые изображения без оптимизации
- Анимации без `prefers-reduced-motion`
- Inline стили (всё через CSS классы)

## 📱 Breakpoints

```css
/* Mobile first */
@media (min-width: 640px) { /* sm: small tablets */ }
@media (min-width: 768px) { /* md: tablets */ }
```

## 🚀 Оптимизация скорости

### Чеклист перед публикацией
- [ ] Все изображения < 100KB
- [ ] WebP формат с JPEG fallback
- [ ] `loading="lazy"` на изображениях ниже сгиба
- [ ] Шрифты с `display=swap`
- [ ] Critical CSS инлайнится в `<head>`
- [ ] Минификация CSS/JS в продакшене

### Целевые метрики
| Метрика | Цель |
|---------|------|
| LCP (Largest Contentful Paint) | < 2.5s |
| FID (First Input Delay) | < 100ms |
| CLS (Cumulative Layout Shift) | < 0.1 |
| Total Page Size | < 500KB |
